# 本論 → どうやって逆転決めたか
- 量を増やし、質を上げることに取り組んだ

## 量: できるだけ勉強時間を確保した
- 勉強をベースに一日を過ごした
  - 勉強する時間を作るんじゃなくて、勉強以外のことをする時間を作るイメージ
  - 1日中勉強勉強する前提で、ご飯とかお風呂の時間を作った
  - 隙間時間にスマホを触らないように単語帳を持ち歩いた
- こうして、この1年で2000時間勉強できた
- ただ、九大の合格者の高校生での平均勉強時間が4000時間と言われているので、量を追求しても効率を上げないと間に合わない状態だった

## 質: 勉強の効率化
1. アクティブリコール: 暗記したものを、何も見ないで0から頭の中で思い出すこと
- ex. 英語 - 英単語
  - とりあえず覚える
  - 単語帳を閉じて、ひたすら覚えた単語と意味を1対1で思い出す
  - 物事を思い出すのに一番負荷が高い方法だから一番記憶に残りやすい
  - 自転車乗ってる時とかご飯食べてるときとか本を開けないときでもできるから、勉強時間を極限まで増やせる
  - 英コミの先生の雑談で聞いて、終盤まで使った
2. 理解すること
- ex. 物理 - 等加速度直線運動
  - 等加速度直線運動では x = v_0*t + 1/2*a*t^2という式が与えられるが、これが実際の動きのイメージと紐づかなかった
  - ただ、位置を微分すると速度、速度を微分すると加速度になるということは感覚的に理解できた
  - だから、そこの感覚と与えられた知識の間を補完してあげることで理解した
3. パターン化: 問題の解法をまとめて共通点を見出す
- ex. 数学
  - FocusGoldの1単元を1周解いたら、2周目は例題の解法と、その解法を選ぶきっかけになった条件/問題文を一覧にまとめた。そして、一覧から共通点を探して解法を更にまとめた
  - たとえば、因数分解なら、「x^2とy^2の入った式 → 字数の低い文字に着目して並べ替える」
  - こういうやつを因数分解の単元全体で集める
  - すると、「2種類以上の文字 → 字数の低い文字に着目して並べ替える」という、因数分解全体で通用する解法が見える
  - これがパターン化で、二次試験で頻出な単元と苦手な単元でやった
4. できることやらない
- 参考書に、問題を解いたときに、理解した上で解けたのかそうじゃないのか日付とともにマークする
- 次開いたときにやらなければならないことが明確で、かつ次に勉強する日の目安もわかりやすい

# 本論
今日は九州大学芸術工学部に合格したので、合格体験談ということで話をさせてもらいます。
自分はもともと映像とかCGとかプログラミングに興味があって、それを全部できる学部を探したときに九大の芸術工学部を見つけて、そこを志望しました。
名前に芸術って入ってますが、入試は普通の理系受験で、英語・数学・物理・化学で受験しました。
難易度も特別低い学部ではなくて、九大の他の理系学部と同じくらいだと思ってもらって大丈夫です。

ただ、自分の受験は全然順調じゃなかったです。
中学のときに起立性調節障害という病気になって、3年間ほぼ学校に行けませんでした。
高校に入ってからも同じような状態が続いて、高1高2は遅刻や欠席をかなり繰り返していて、正直、留年ギリギリの状態でした。
なので勉強時間をほとんど取れていなくて、中学レベルから基礎が抜けている状態からスタートしました。

勉強を始める前は、模試はずっとD判定で、共通テスト模試も600点台くらいでした。
そこから本格的に勉強を始めて、夏前の共通テスト模試で750点くらいまで上がって、
11月の九大の冠模試では合格点を取れて、A判定で学部内2位でした。
最終的に共通テストは838点まで伸びて、そのまま合格することができました。

どうやってここまで伸ばしたかという話をすると、
自分はとにかく「量を増やすこと」と「質を上げること」の両方を意識して勉強しました。

まず量についてですが、できるだけ勉強時間を確保することを意識しました。
勉強する時間を作るというより、勉強を前提に生活して、それ以外の時間を作るイメージで過ごしていました。
一日の中でまず勉強をして、その中にご飯とかお風呂とかを入れる感じです。
隙間時間にスマホを触らないように単語帳を持ち歩いたりして、とにかく勉強時間を増やしました。

その結果、この1年間でだいたい2000時間くらい勉強しました。

ただ、九大の合格者は高校3年間で4000時間くらい勉強していると言われていて、
自分は中学や高1高2でほとんど勉強できていなかったので、
量を増やすだけでは追いつかない状態でした。

なので、勉強時間を増やすだけじゃなくて、効率もかなり意識して勉強しました。

1つ目はアクティブリコールです。
これは、覚えたことを何も見ないで思い出す勉強法です。

例えば英単語なら、普通は単語帳を見ながら覚えると思いますが、
ある程度覚えたら単語帳を閉じて、頭の中で意味を思い出すようにしていました。

思い出す作業は負荷が高いので、その分一番記憶に残りやすいです。
自転車に乗っているときとか、ご飯を食べているときとか、本を開けないときでもできるので、
勉強時間をかなり増やすことができました。

これは英コミの先生の雑談で聞いた方法で、最後までずっと使っていました。

2つ目は、できるだけ理解して覚えることです。

例えば物理で等加速度直線運動をやったときに、
公式をそのまま覚えるだけだと全然頭に入らなかったんですが、
位置を微分すると速度になって、速度を微分すると加速度になる、という関係は感覚的に理解できました。

なので、そういう自分が分かる感覚と、教科書に書いてある知識をつなげるようにして、
できるだけ納得してから覚えるようにしました。

3つ目はパターン化です。
問題の解き方をまとめて、共通点を探すようにしていました。

例えば数学でFocusGoldをやっていたときに、
1周目で普通に解いたあと、2周目では解法と、どういう条件のときにその解法を使うかをまとめました。

例えば因数分解なら、
x²とy²が出てきたら、文字の少ない方に注目して並べ替える、みたいなルールを作って、
それを単元全体で集めていきました。

すると、その単元で共通して使える解き方が見えてきて、
問題を見たときに迷う時間がかなり減りました。

これは二次試験でよく出る単元とか、苦手な単元を中心にやっていました。

4つ目は、できることをやらないことです。

参考書を解いたときに、
理解して解けた問題と、分からなかった問題を日付と一緒にマークしておいて、
次に開いたときに、やるべきところだけをやるようにしていました。

そうすると、何をやればいいか迷わないし、
同じところばかりやることもなくなるので、効率よく進めることができました。

以上が、自分がやっていた勉強法です。
6年分くらいの内容を1年に圧縮するつもりで勉強しました。

ただ、やっていること自体は特別なことじゃなくて、
量を増やして、効率を上げて、当たり前のことを続けただけです。

自分も1年前は全然できていなかったので、
今の時点で勉強できていなくても、これから始めれば間に合うと思います。

勉強を始めるのが1日遅れると、その分解ける過去問が1年分減ると思ってやっていました。

なので、まずは好きな科目からでもいいので、
ちょっとずつでも始めてみてほしいと思います。
