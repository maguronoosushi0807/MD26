# 複素数平面
## 基本計算
### 一般
- **計算が複雑になりがち**
- ベクトルにおいて2乗したり $\overrightarrow{AP} = \overrightarrow{OP} - \overrightarrow{OA}$ を使うように、複素数では頻繁に共役な複素数を扱う。
- $f(\alpha)=0$ のとき、 $f(\bar{\alpha})=0$より、実数係数の方程式において、$\alpha$ が解となるとき、$\bar{\alpha}$ も解となる
- 共役の証明 → z=a+biとおき成分で
- iの乗算・除算 → $\pm$ の順に注意する
- zが実数/虚数になる → z=(実部)+i(虚部) と分離し、=0とする

### 極形式 $z=r(\cos\theta+i\sin\theta)$
- $i\sin\theta$ の係数は正でなければならない
- 極形式において、半径 $r$ は正でなければならない
- 有名角でないものの極形式を求める → 虚数の積・商の公式を用いる
- 3項以上の極形式 → 実部・虚部に分け、まずは大きさを求める
- 極形式で表されたものが正の数 → $\theta=2n\pi$
- $\arg\alpha = \theta$ → $\arg\alpha^2 = 2\theta$
- $\alpha = r_1(\cos\theta_1+i\sin\theta_1), \beta = r_2(\cos\theta_2+i\sin\theta_2)$ のとき
  - $\alpha\beta = r_1r_2{\cos(\theta_1+\theta_2)+i\sin(\theta_1+\theta_2)}$
- 二次方程式の解を極形式で → 普通に解いてから極形式になおす
- 散らばった三角関数をまとめる → 和積の公式
### ド・モアブルの定理の利用
- $z^n$ に関する式 → まずは $z=r(\cos\theta+i\sin\theta)$ とおく
- $a$ の $n$ 乗根 → $r^n(\cos (n\theta)+i\sin (n\theta))=a$ とおく
- (複素数)$^n$ → ド・モアブルの定理
  - $(\cos\theta+i\sin\theta)^n = \cos(n\theta)+i\sin(n\theta)$

### 計算量を減らす
  - $\pm$ → 場合分けせずそのまま処理する
  - $z$ を求めろと言われていない → $z^2$ のまま処理する
  - 割り算 → 符号の規則性、対称性を考えて共通因数でくくる
- なす角 $\theta$ → $\arg\frac{\gamma-\alpha}{\beta-\alpha} = \theta$
- $|\alpha|=1$ → $\bar{\alpha}=\frac{1}{\alpha}$
- 対称式 → 同様に～ で楽にする
- 最終手段 → $a+bi$ の置換
- $z$ の置換
  - $z=r(\cos\theta+i\sin\theta)$
  - $z=a+bi$
- $\alpha$ について整理してうまくいかない → $\beta, \gamma$ を順に試す
- 2式が与えられた → 1文字消去

## 複素方程式
**基本方針: 両辺共役な式をつくり連立する**
- 最終手段 → $a+bi$(そのまま計算する)
- 与式をかけて回転させると一致するか疑う
- 複素数漸化式 → $\alpha^n,  (\alpha=$(複素数)$)$ を最優先で消す
- 三角関数を求める → 積の公式と極形式で計算したものを比較

## 実数 / 虚数
- **実数条件 / 虚数条件**
  - $z$ が実数 → $z=\bar{z}$
  - $z$ が純虚数 → $z=-\bar{z}, z\ne0$ 
- zは実数か純虚数か？ → 実数条件 / 虚数条件 で評価する
- 実数/純虚数であることを証明 → 上のどちらかの形を作る
- **隠れた実数条件**
  - $z$ が正の数/負の数
  - $z$ の範囲
- **問題文を見たらまずやること**
  - zは実数である → とりあえず $z=\bar{z}$ を把握
  - zが純虚数の場合も同様
  - zが虚数ならどちらにも当てはまらないため放置

## $|z|$ の扱い
- $|z|$ に関する等式
  - $|z^n| = |z|^n$
  - $|z|=|\bar{z}|$
- $|z|$ を求める → まず2乗して以下のどちらかを利用
  - $|z|^2 = z\bar{z}$
  - $|z|^2 = a^2+b^2$
- $|z|^2$ の等式の証明 → $|z|^2=z\bar{z}$ を用いる
- $|z|=1$ など、$|z|$ の条件が与えられている → $\bar{z}=\frac{1}{z}$ を利用
- $z\bar{z}$ → $|z|^2$ を利用
- $|z|$ の最大・最小 → $|z|$ の中の項をできるだけ減らす
  - まずは図形的に考える
  - 駄目なら代数的に解析

## 一直線 / 垂直
- $\alpha, \beta, \gamma$ が一直線上にある ⇔ $\frac{\gamma-\alpha}{\beta-\alpha}$ が実数
- $\alpha, \beta, \gamma$ が垂直 ⇔ $\frac{\gamma-\alpha}{\beta-\alpha}$ が純虚数 $(\ne0)$


## 複素数平面
**最優先で回転・拡大を考える**
- 想像しにくい場合 → 原点に動かしてから回転を考える発想
### 点編
- 点 $\alpha$ を中心に $\beta$ を $\theta$ 回転させた点 $z$ → $z-a = (\cos\theta+i\sin\theta)(\beta-\alpha)$
- どのような点か？ → 同じ形を作る
### 図形編
- どのような三角形か？ → 1点を基準にもう1点を回転・距離を拡大させ、のこりの1点に重ねる事を考える
- 原点中心のABCが正三角形を示す → ABの中点をMとすると、Mと逆側に2OMの長さの位置にCが存在する
- 直角三角形のみでは一意に定まらないため、辺の比を記す
- 直角の証明 → なす角でなく回転で考える
- 平行四辺形 → ベクトルで考えて立式する

## 複素数の表す図形を求める
- **基本解法 3つ**
  - $z$ について解いて、zの式に放り込む
  - 複素数的に見て、$z$ とどんな関係があるかを考える
  - $z=a+bi$ としてゴリ押し
- 図形の性質から考えて、目的の点を回転などで表す
- なす角で考える
- 単位円上の $\alpha, \beta, \gamma$ → 垂心 $H=\alpha+\beta+\gamma$
- $z$ と $\bar{z}$ を関係づけることができない → $\bar{z}$ を最優先で消す
- 内心 → 頂点と内接円の接点の辺の長さの性質から回転させる
- 円上の複素数 → 回転させる
- 複素数が分子にある場合、実数と同様に $\neq0$ を確認する

## 存在範囲を求める
- $\theta$ の範囲が切れる → $2\pi$ をひいたときにもとのやつに戻ってくればいい
- $|z|, \arg{z}$ の最大・最小 → 図形的に考える
- (～) の表す範囲 → (～) $=z$ とおいて、与えられた条件式から $z$ の形を作る
- $z$ に大小関係が存在する → $z$ は実数
