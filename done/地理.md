# 自然環境
### 火山
- 火山 → 新期造山帯のプレート境界 or アフリカ大地溝帯
- 火山噴出物 → 隙間が多く、水はけが良い
- カルデラ → 北海道
### 氷河
- 山岳氷河 → 標高が基準を超えるかで判断 赤道付近:5000m-6000m/温帯:3000m-4000m
- 氷河 → 夏に融解して川の流量 増加
- アルプス山脈 → 氷河
### 自然災害
- 震源の深さ
  - 直下型 → 浅い
  - 海溝型 → 深い
- 中米 → ハリケーンが襲来する
- 洪水の発生時期 → 川の流量変化の多い高緯度帯に着目
### 雨温図
- 気温/降水量の図の読み方
  - 1.気温 → 北半球/南半球/赤道に分類
  - 2.降水 → 内陸/海岸に分類
  - 3.Csなど、問題によっては位置まで推論
- 高山地域の気温の較差 → 年較差 < 日較差
- 日照時間 → 雨の量から判断
### 植生
- 土壌/植生 → 気候
  - プレーリー・パンパ → 長草草原
  - レグール(インド) → 溶岩台地の玄武岩
- 森林の有無 → 気温+降水量
- 植物の分解 → CO2とH2Oになると考える
### 知識
- 変動帯 = 新期造山帯
- 高低 → プレート境界
- 熱帯収束帯の動く幅は年によってまちまち
- 卓越風 = その時期に一番吹きやすい風
- エルニーニョ現象 = 原因不明の貿易風の弱化により湧昇流が減ることで起こる、ペルー沖の海水温の異常上昇
- 積雪 → 12月から(北海道は例外的にもっと早い)
