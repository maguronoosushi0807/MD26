# 本論 → どうやって逆転決めたか
- 量を増やし、質を上げることに取り組んだ

## 量: できるだけ勉強時間を確保した
- 勉強をベースに一日を過ごした
  - 勉強する時間を作るんじゃなくて、勉強以外のことをする時間を作るイメージ
  - 1日中勉強勉強する前提で、ご飯とかお風呂の時間を作った
  - 隙間時間にスマホを触らないように単語帳を持ち歩いた
- こうして、この1年で2000時間勉強できた
- ただ、九大の合格者の高校生での平均勉強時間が4000時間と言われていて、勉強を始めた段階から量を追求しても効率を上げないと間に合わない状態だった

## 質: 勉強の効率化
1. アクティブリコール: 暗記したものを、何も見ないで0から頭の中で思い出すこと
- ex. 英語 - 英単語
  - とりあえず覚える
  - 単語帳を閉じて、ひたすら覚えた単語と意味を1対1で思い出す
  - 物事を思い出すのに一番負荷が高い方法だから一番記憶に残りやすい
  - 自転車乗ってる時とかご飯食べてるときとか本を開けないときでもできるから、勉強時間を極限まで増やせる
  - 英コミの先生の雑談で聞いて、終盤まで使った
2. 理解すること
- ex. 物理 - 等加速度直線運動
  - 等加速度直線運動では x = v_0*t + 1/2*a*t^2という式が与えられるが、これが実際の動きのイメージと紐づかなかった
  - ただ、位置を微分すると速度、速度を微分すると加速度になるということは感覚的に理解できた
  - だから、そこの感覚と与えられた知識の間を補完してあげることで理解した
3. パターン化: 問題の解法をまとめて共通点を見出す
- ex. 数学
  - FocusGoldの1単元を1周解いたら、2周目は例題の解法と、その解法を選ぶきっかけになった条件/問題文を一覧にまとめた。そして、一覧から共通点を探して解法を更にまとめた
  - たとえば、因数分解なら、「x^2とy^2の入った式 → 字数の低い文字に着目して並べ替える」
  - こういうやつを因数分解の単元全体で集める
  - すると、「2種類以上の文字 → 字数の低い文字に着目して並べ替える」という、因数分解全体で通用する解法が見える
  - これがパターン化で、二次試験で頻出な単元と苦手な単元でやった
4. できることやらない
- 参考書に、問題を解いたときに、理解した上で解けたのかそうじゃないのか日付とともにマークする
- 次開いたときにやらなければならないことが明確で、かつ次に勉強する日の目安もわかりやすい
